# Employee Dashboard React App

A responsive **Employee Management Dashboard** built with **React.js**, allowing admin and employee users to manage tasks efficiently.  

---

## 🚀 Features

### Admin
- Login as admin.
- View all employees and their task counts (New, Active, Completed, Failed).
- Assign tasks to employees.

### Employee
- Login as an employee.
- View personal task statistics.
- See tasks categorized by status: New, Active, Completed, Failed.

### Tasks
- Create new tasks with:
  - Title
  - Description
  - Date
  - Category
  - Assign to specific employee
- Automatic task counts update for each employee.

### Responsive Design
- Fully responsive layout for desktop and mobile screens.
- Modern UI with colored task status cards.

---

## 🛠 Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **State Management:** React Context API, useState
- **Storage:** LocalStorage (for demo purposes)
- **Routing:** React Router (optional future enhancement)
